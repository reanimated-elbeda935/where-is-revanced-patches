# Repository Status

## What happened?

A few days ago, a third party submitted a DMCA takedown notice to GitHub concerning this repository.

As required by law, GitHub forwarded the complaint to us and requested that we either comply with the notice or submit a counter-notification. This is part of GitHub’s standard process to maintain its safe harbor protections under copyright law.

## Why is the repository unavailable?

GitHub processes counter-notifications only after a repository has been taken down. Once a takedown occurs, the complainant is given a legally mandated 14-day period to respond with legal action or accept the counter-notification.

During this period, the repository remains unavailable. This is a standard precaution taken by GitHub to avoid potential liability.

## Our response

After reviewing the complaint, we have determined that it has no legal basis.

We have therefore submitted a counter-notification to GitHub, which has been forwarded to the complainant.

## Where can I find the project now?

Development continues uninterrupted on our mirror repository:

👉 [https://gitlab.com/revanced/revanced-patches](https://gitlab.com/revanced/revanced-patches)

## What happens next?

We expect the repository to be reinstated after the 14-day waiting period, as the complaint lacks legal merit.

Once reinstated, development will continue here as usual.
